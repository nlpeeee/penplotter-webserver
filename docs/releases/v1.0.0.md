# PCP V1.0.0 release record

PCP V1 is frozen at commit
`f883f63b420691c5c6be3bbd435fd8b509367d3f` and annotated tag `v1.0.0`.
The maintenance branch is `release/v1`. Neither the tag nor its source
baseline is to be moved.

## Included production features

- Exact SVG and HPGL path preview.
- Millimetre scaling, placement, rotation, mirroring, and roll validation.
- Canonical 40-unit/mm HPGL generation and ordered path simulation.
- Preflight, deterministic copies/layout, vinyl cutting aids, profiles, and
  per-cutter calibration.
- Self-contained saved projects with immutable revisions and exact recuts.
- Persistent spool-backed queue, job history, cancellation, and guarded USB
  serial reset.
- PCP branding with the original WebPlot MIT attribution retained.
